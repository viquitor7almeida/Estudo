# Repositorio de Estudos: Fundamentos de Programacao

Este repositório reúne uma serie de exercícios e implementações focadas nos pilares da programação e orientação a objetos. O objetivo principal e manter os fundamentos sempre disponíveis para consulta rápida, garantindo que os conceitos essenciais nao "atrofiem" com o tempo para os avançados e também sirvam de guia para quem esta iniciando na jornada.

## Propósito para avançados

Programar sistemas complexos muitas vezes nos faz esquecer a base. Este espaço serve para quebrar isso e trazer conceitos gerais e cruciais de java.

## Propósito para iniciantes

Servir como um motor e um guia de estudos no core da linguagem java.

## Como utilizar

Cada diretório aborda um conceito específico (como lógica matemática, estruturas de repetição ou manipulação de objetos). Contendo um arquivo de texto/resumo para leitura e um arquivo de código comentados de forma a explicar não apenas o "que" o codigo faz, mas "como" ele se comporta por baixo do pano.

## Filosofia

A leitura constante do básico é o que permite a compreensão do avancado. Este repositório e um registro de que a base deve ser revisitada sempre para evitar a degradação do raciocínio lógico e técnico.
